### Code-Beispiele

Code-Beispiele müssen nicht zwingend in einem eigenen Kapitel sein. Sie können auch durchaus im Inhalt sein.

    System.out.println("Hello Java");
  }
}
```
</details>

  #### C#</summary>

}
```
</details>
